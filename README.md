# 🚀 TaskFlow Pro - Advanced To-Do Manager

A modern, feature-rich, and fully responsive To-Do List web application built using **HTML5, CSS3, and Vanilla JavaScript**. TaskFlow Pro helps users efficiently manage daily tasks with advanced filtering, local storage persistence, dark mode support, and a beautiful glassmorphism-inspired UI.

🌐 **Live Demo:**  
https://prawin-taskflow-pro-app.vercel.app

---

# 📸 Project Overview

TaskFlow Pro is designed to provide a clean and productive task management experience with modern UI/UX principles and powerful client-side functionality.

### ✨ Highlights

- Modern Glassmorphism UI
- Dark & Light Theme Support
- Fully Responsive Design
- Local Storage Persistence
- Advanced Search & Filtering
- Full CRUD Functionality
- Dynamic DOM Manipulation

# 🚀 How to Run Locally

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/PrawinKumar514/taskflow-pro.git
```

### 2️⃣ Navigate to the Project Directory

```bash
cd taskflow-pro
```

### 3️⃣ Open the Project

Open the project folder in:

- Visual Studio Code
- Sublime Text
- Any code editor of your choice

### 4️⃣ Launch the Application

Open:

```text
index.html
```

directly in your browser

**OR**

Use VS Code Live Server:

- Right-click `index.html`
- Click **Open with Live Server**

### 5️⃣ Start Managing Tasks

You can now:

- ➕ Create Tasks
- ✏️ Edit Tasks
- 🗑️ Delete Tasks
- ✅ Mark Tasks as Completed
- 🔍 Search Tasks
- 🎯 Filter Tasks
- 🌙 Switch Between Dark & Light Mode

All data will be automatically saved using Local Storage.
